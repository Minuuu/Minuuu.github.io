OpenCv를 시작하기 위한 프레임워크 셋업

프레임워크 셋업

  • 모든 설치 및 업그레이드는 cmd로 한다.

  • 관리자 권한으로 cmd를 실행

    Python 3.7.3 설치 or 업그레이드

  • 오류

    1. pip ssl이슈 처리: https://surpreem.com/팁-파이썬-패키지-설치할-때-ssl-인증-오류-해제-방법/

OpenCv4 설치

  • 오류
    1. dlib 설치시 CMake가 미리 설치 되어있어야 한다.
      참고사이트: https://westwoodforever.blogspot.com/2013/04/cmake-windows.html
    2. dlib 설치시 setup.py관련 에러 발생이 경로에 한글명이 들어갔나 확인해 보자! 한글명이 있으면 한글인식을 못해 에러가 발생
    3. dlib 설치시 오래 걸리니 참을성 있게 기달리자…

matplotlib 설치

  • matplotlib는 그래프 라이브러리

mahotas

  • Numpy배열들을 이용하는 OpenCV류의 라이브러리
  • Mahotas interface를 OpenCV와 연동하면 유용

SciPy Stack

  • Numpy는 대용량, 다차원 배열을 처리하는데 효과적인 Python라이브러리. Numpy를 이용하면 다차원의 이미지를 효과적으로 연산하고 표현가능.
  • 이미지 및 영상 관련 ML의 대부분의 Numpy를 통해 처리
  • 실시간 영상처리를 지향하는 OpenCV가 2009년부터 Numpy가 지원되면서 부터 개발자들의 OpenCV를 이용한 연구성과및 퍼포먼스가 대폭 좋아짐
  • SciPy는 Numpy를 포괄하는 영역까지 처리

  • Numpy 1.16.3 설치

  • SciPy 1.3.0 설치

Deeplearning 라이브러리

  • scikit-learn 설치

    • scikit-learn은 ML의 라이브러리, 이미지나 영상 전용은 아니지만, Clustering, vector quantization, classification models 생성에 유용하게 사용가능

  • scikit-image 설치

    • 정적인 이미지 학습시 사용하는 라이브러리로, 샘플 셋을 학습시킬 때 매우 유용

  • Keras & Pytorch & Tensorflow 비교

    • Keras

      • Top2
      • Tensorflow와 Theano에서 돌아가는 하이레벨 API
      • 적은 데이터셋에 효과적

      • 다양한 백엔드 언어를 지원

      • 설치전에 numpy, six, h5py, keras-applications, scipy, keras-preprocessing,pyyaml 설치 필요

    • Tensorflow

      • Backed by Google(Top2)
      • support C/C++/Java/Go
      • 데이터셋이 많을때 효과적
      • 퍼포먼스가 좋음
      • Product Oriented 프로젝트에 좋음
      • 특히 객체 인식에 효과적

    • Pytorch

      • Numpy와 SciPy를 자유롭게 사용가능
      • 학습 훈련시간이 적게 듦
      • 작은 프로젝트에서 좋음
